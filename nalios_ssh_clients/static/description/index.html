SSH Manager

SSH Manager lets you manage all your SSH Connection, connect to them and create routines with commands to send. /!\ it does not support interactive commands like vim /!\

Compatible with Odoo SH and On Premise versions


Configuration steps (view video below)

  • 1/ Install the module.
  • 2/ Give access in User access rights
  • 3/ Create SSH connections
  • 4/ Connect, or create routines to send to your SSH Connection

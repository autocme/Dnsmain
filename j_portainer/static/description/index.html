Portainer Integration for Odoo

Manage your Docker environments directly from Odoo

This module provides seamless integration with Portainer Community Edition 2.27.4 LTS, allowing you to manage and monitor your Docker containers, images, networks, volumes, stacks, and environments directly from your Odoo installation.

Key Features

  • Connect to multiple Portainer servers
  • Secure API Key authentication
  • Manage all your Docker environments
  • View and control containers (start, stop, restart, remove)
  • Manage Docker images
  • Control Docker volumes and networks
  • Deploy from application templates
  • View and manage stacks
  • Detailed resource information

Easy Synchronization

Keep your Portainer resources in sync with Odoo using a simple wizard. Choose to sync all resources or just specific ones like containers, images, or volumes.

Technical Information

Requirements
  • Portainer Community Edition 2.27.4 LTS or compatible
  • API Key access to your Portainer server
  • Network connectivity from Odoo to Portainer
